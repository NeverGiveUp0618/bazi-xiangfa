# 八字象义图谱维护说明

直接双击 `index.html` 可以打开网站。

文件分工：

- `index.html`：页面结构，通常不用改。
- `styles.css`：视觉样式，改颜色、布局、间距时改这里。
- `data.js`：象义内容库，后续主要维护这个文件。
- `app.js`：展开、搜索、详情联动等交互逻辑，通常不用改。

当前内容规模：

- 阴阳五行：6 个节点
- 资料摘录映射：10 个节点
- 十神象义：10 个节点
- 天干象义：10 个节点
- 地支象义：12 个节点
- 干支关系：12 个节点
- 常用神煞：12 个节点
- 纳音象义：30 个节点
- 宫位岁运：8 个节点

合计：9 个体系，110 个节点。

表达风格：

- 110 个节点均保留专业象义。
- 110 个节点均已补充通俗解释层，如“大白话、像什么、优点、风险、提醒、方向”等。
- 目标是同一节点既能给初学者看懂，也能保留后续继续深化的专业结构。
- 右侧详情的“取象说明”会自动显示大白话，并解释当前词为什么能归到对应分支。
- 右侧“关联与重叠”会自动计算象义交集，显示共享词、重叠强度和可跳转节点，用来观察哪些取象互相印证。

资料摘录说明：

- 已从 `source_extracts/六爻取象1.txt` 中提取可迁移到八字的六亲象义：父母爻对应印星、官鬼爻对应官杀、兄弟爻对应比劫、妻财爻对应财星、子孙爻对应食伤。
- 已从 `source_extracts/张泽华-万物类象.txt` 中提取八卦万物类象，并折算为五行辅助象：乾兑补金象、震巽补木象、离补火象、坎补水象、坤艮补土象。
- 已从 `source_extracts/纳音类象.txt` 中补充 30 个纳音的白话气质、优点、风险和适合方向。
- 已排除六神、世应、具体六爻卦例等六爻专属内容。

新增节点时，在 `data.js` 对应体系的 `nodes` 数组里增加一段：

```js
{
  id: "unique-id",
  title: "节点名称",
  type: "十神 / 天干 / 地支 / 关系 / 纳音",
  core: ["核心义一", "核心义二"],
  branches: {
    "顶层词": ["分支词一", "分支词二"],
    "人物": ["人物象一", "人物象二"],
    "事物": ["事物象一", "事物象二"]
  },
  rules: ["取象规则一。", "组合条件二。"],
  relations: ["关联节点一", "关联节点二"]
}
```

建议后续按这个顺序继续扩充：

1. 给十神增加“组合变化”：如官印相生、伤官见官、食神制杀、财破印、比劫夺财。
2. 给天干地支增加“不同季节取象”：如甲木在寅月、申月、子月的差异。
3. 给干支关系增加“宫位落点”：冲年、冲月、冲日、冲时分别如何取象。
4. 给纳音增加六十甲子的逐柱解释。
5. 增加出处、门派差异、正反象、应期条件和案例。
